# MovieMate

## 목적

최신 인기 영화와 영화에 대한 정보 제공의 웹 어플리케이션을 만들기 위해 구현하였습니다. 외부 API를 이용한 리액트 프로젝트 실습의 목적 또한 있습니다.

##

- Home page에 인기 영화 목록이 나옵니다. More Movies 버튼을 클릭하면 더 많은 영화를 조회할 수 있습니다.
- About Us를 클릭하면 About Us page로 이동하며 MovieMate에 대한 정보를 확인할 수 있습니다.
- 영화를 클릭하게 되면 MovieDetail page로 넘어가게 됩니다.
- MovieDetail page에는 영화에 대한 정보가 제공되며, Watching Actor를 클릭하게 되면 영화 출연배우에 대한 정보를 얻을 수 있습니다.

## 사용 기술

React, Styled-components, React Router DOM, Ant Design, TMDP API
